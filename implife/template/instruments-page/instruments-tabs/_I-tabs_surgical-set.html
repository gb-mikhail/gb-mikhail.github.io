Хирургический набор

Компактный и удобный набор содержит оптимальное количество инструментов для работы с одно- и двухэтапными имплантатами на хирургическом и ортопедическом этапах.
Бокс изготовлен из нержавеющей стали предназначен для хранения и стерилизации инструмента. Может подвергаться термической стерилизации не выше 134 С°.
